## Issues Discovered

No application or contract defect was discovered during the smoke run. The backend started successfully, health passed, and all smoke endpoints returned successful responses.

One environment issue did appear during execution from the default sandbox: loopback access to `127.0.0.1:8000` was not reachable from the sandbox even though Uvicorn had started. The measured run was therefore executed outside the sandbox so the real local runtime could be reached. This was an environment/runtime isolation issue, not an API defect.
